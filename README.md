<!-- omit in toc -->
## Content

## What does it do?
This extension configures TYPO3 to use `Mailjet` for sending emails. Also it comes with a database table, in which all email sending actions are inserted, so that you can see a statistics about the email sending process.

## Installation
The extension can easily be installed using composer:
```bash
composer install worlddirect/mailjet
```
